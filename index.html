Pennymine Project

The goal of this project is to use data mining techniques to predict whether a penny stock will close a winner or a loser the next day. At the time of writing, the project is halfway complete. Various penny stocks were chosen based on a set of criteria. Historical price data was collected, and some machine learning algorithms were tested on the data. The results showed that historical price data alone is not enough to predict stock performance. Next steps are to collect promotional data from online forums and newsletters and combine it with previously collected price data in an attempt to produce better results. Take a look at the Google Doc to learn more about this project and things that have been tried so far.
